Resumo

A anlise de varincia um tpico extenso e bastante complexo da estatstica. Aqui introduziremos as formas mais simples de comparao de trs ou mais mdias com somente um fator de comparao (One-way ANOVA e Kruskal-Wallis). Adicionalmene, abordaremos noes sobre a anlise de varincia de dois fatores e suas diversas possibilidades, por meio de alguns exemplos prticos.

Anlise de Varincia: One Way

Para executar a anlise de varincia pode-se entrar ou estruturar os dados de duas maneiras:

  • em colunas em que o conjunto de dados para cada situao experimental ou fator est organizado em colunas, como neste exemplo; e

  • em duas colunas de resposta e fator, em que os dados de medida esto numa coluna denominada resposta e a situao experimental que os gerou identificados na coluna fator.

Condies para executar One-way ANOVA

Homoscedasticidade

Vamos verificar se existe homogeneidade das varincias dos dados em anlise, pois ANOVA s pode ser usada se as varincias das medidas dos grupos forem similares entre si, alm de a amostra provir de uma populao com distribuio normal para a varivel em estudo.

Como sempre, devemos estabelecer as hipteses nula e alternativa:

$H_0 : \sigma_{1}^2 = \sigma_{2}^2 = \sigma_{3}^2$
$H_1 : \sigma_{1}^2 \ne \sigma_{2}^2 \ne \sigma_{3}^2 $

No h uma forma boa de escrever a hiptese alternativa. Caso rejeitemos a hiptese nula em favor da alternativa, significa que assumiremos que pelo menos uma das varincias destoa das demais, e no que todas so diferentes entre si.

Adotaremos nvel de significncia de 5%.

Para aqueles que se lembram da Aula de Teste t, bom ressaltar que existe uma rotina distinta para o teste de homogeneidade das varincias aplicada a ANOVA, pois neste caso desejamos comparar mais que duas varincias simultaneamente. Observe porm que a motivao de testagem de igualdade de varincia no teste t no-pareado distinta. No teste t, o resultado da testagem de varincias determina a expresso do erro-padro da mdia das diferenas, pois se as varincias forem iguais necessrio o clculo da varincia conjugada ($ S_{0}^2 $, vide aula terica sobre teste t), mas o teste t continua aplicvel. No caso presente, caso rejeitemos a hiptese nula, o teste ANOVA deixa de ser indicado, e precisaremos aplicar seu equivalente no paramtrico, teste de Kruskal-Wallis, que veremos adiante.

Para realizar o teste de homogeneidade de varincias, execute:

$\ $Estatsticas -> Varincia -> Teste de Barlett...

$\ $ Statistics -> Variances -> Bartlett's test...

No h opes, e como temos apenas uma varivel qualitativa nominal (Grupo) e uma quantitativa contnua (Valor), o Rcmdr j as mostra. Clique Ok para obter:

Com $ p = 0,4913 $, para $ \alpha = 0,05 $, aceitamos a hiptese nula. Conclumos que as varincias dos dados dos grupos X1, X2 e X3 so todas similares entre si e, portanto, podemos aplicar o teste de ANOVA para testar a igualdade das mdias dos trs grupos estudados.

Caso tivssemos rejeitado $ H_0 $ concluiramos que pelo menos uma das varincias ($ \sigma_1 ^2 $, $ \sigma_2 ^2 $ ou $ \sigma_3 ^2 $) difere das demais e no poderamos aplicar o teste de ANOVA. Optaramos pelo seu equivalente, no paramtrico, [teste de Kruskal-Wallis], descrito adiante.

Algo para testar (uso avanado)

Voltando anlise de uria e creatinina com cinco grupos de faixa etria, note que, diferentemente da implementao do ANOVA, no RCmdr no h como automatizar a aplicao do teste de Mann-Whitney aos pares.

D para apelar para a linha de comando. Em primeiro lugar, precisamos corrigir o valor de alfa. A mais simples de todas a correo de Bonferroni, que divide o valor de $ \alpha $ pelo nmero de pares (combinatria do nmero de grupos tomados 2 a 2). A frmula da combinatria :

$$ _{x}^{n}\textrm{C} = \frac{n!}{\left ( n-x \right )! \cdot x!} $$

No exemplo em que temos 5 grupos que, tomados em pares o resultado...

$$ _{2}^{5}\textrm{C} = \frac{5!}{\left ( 5-2 \right )! \cdot 2!}= $$

... pode ser preguia para fazer a conta. Podemos (como j sabem) usar RCmdr como calculadora. Como existe a funo factorial($\dot$), experimente

factorial(5) / (factorial(5-2) * factorial(2))

que lhe retornar

[1] 10

O valor corrigido, que usaremos para comparar ser $ \alpha_{_{\text{Bonferroni}}} = \frac{\alpha}{10} = 0.005 $.

Tambm j vimos a sintaxe que o RCmdr aplica para o teste de Mann-Whitney. No conjunto de dados que temos, ao tentarmos:

$\ $Estatsticas -> Teste No-Paramtrico -> Teste de Wilcoxon (teste 2 amostras)...

$\ $Statistics -> Nonparametric tests -> Two-sample Wilcoxon test...

o nico agrupamento disponvel SEXO. Isto seria bem esperado, pois este teste usa, para definir grupos, uma varivel qualitativa nominal que tenha somente dois estados (no caso masculino e feminino). Experimente ver se UREIA difere entre os sexos. Obter:

O primeiro, UREIA_wilcoxon.R contm:

faixas <- c("0 a 20","21 a 40","41 a 60","61 a 70","> 70")
num_grupos = length(faixas)
combinacoes <- factorial(num_grupos) / (factorial(num_grupos-2)*factorial(2))
alfa = 0.05
alfa_bonferroni = alfa/combinacoes

print (paste ("alfa corrigido = ",alfa_bonferroni, " combinacoes = ", combinacoes))
for (par1 in faixas)
{
  for (par2 in faixas)
  {
    if (par2 > par1)
    { 
        p <- wilcox.test(Exames$UREIA[Exames$FAIXA_ETARIA==par1 | Exames$FAIXA_ETARIA==par2] ~ 
        + Exames$FAIXA_ETARIA[Exames$FAIXA_ETARIA==par1 | Exames$FAIXA_ETARIA==par2], alternative="two.sided")
        if (p[3] < alfa_bonferroni)
        {
            significante = " (H1) ... medianas diferentes"
        }
        else
        {
            significante = " (H0) ... medianas iguais"
        }
        print (paste (par1," vs. ",par2," p=",p[3],significante))
    }
  }
}
numSummary(Exames[,"UREIA"], groups=Exames$FAIXA_ETARIA, 
  statistics=c("mean", "sd", "quantiles"), quantiles=c(0,.25,.5,.75,1))
